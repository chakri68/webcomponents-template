# Vite, Lit and Storybook for standalone web component development starter pack

Insipred by [this article](https://leon.id/articles/web-components/2022-02-vite-lit-storybook)

## Install

```bash
yarn
```

## Develop

```bash
yarn dev
```

## Storybook

```bash
yarn storybook
```

## Build

```bash
yarn build
```
